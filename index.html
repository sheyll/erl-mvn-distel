erl-mvn-distel by sheyll

Smoother integration erlang-plugin for maven with distel and emacs

After the availability of the current (2.0.0) version of the maven erlang plugin, I was missing some kind of IDE support for working with mavenized erlang projects. I wrote this emacs mode in order to compile erlang buffers and run eunit tests. Because dependency management is done with maven, all dependencies can automatically downloaded and uploaded to an erlang node by maven. This emacs mode manages erlang nodes to which all code is uploaded either via maven or directly by distel on buffer modifications. This mode keeps track of sources that are part of maven erlang projects and automatically starts required erlang nodes for each project.

Dependencies

This depends on: * maven erlang plugin version 2.0.0 * most recent distel from github

Install

The ~/.emacs file should contain the statements necessary to load the erlang and distel modes. Add the following lines _after_ Distel and Erlang mode: ;Erlang Maven and Distel intergration (add-to-list 'load-path "") (require 'erl-mvn) (erl-mvn-setup)
